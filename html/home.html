🤖 Agentic AI — Multi-Agent Systems

Welcome to this comprehensive educational resource on Agentic AI and Multi-Agent Systems. This course is designed for working software engineers (5+ years) who want to master the design and implementation of production-grade AI agent systems.

Audience: Senior engineers transitioning to Generative AI  |  Format: Self-paced reading + hands-on code labs

⚡ Course Setup — Install everything & run your first agent 
# 1. Create a virtual environment
python -m venv .venv
source .venv/bin/activate            # macOS / Linux
# .venv\Scripts\activate             # Windows PowerShell

# 2. Install all course dependencies (one-time)
pip install langchain langchain-openai langchain-community langgraph \
            openai faiss-cpu fastapi "uvicorn[standard]" streamlit \
            langsmith pydantic python-dotenv

# 3. Set your OpenAI API key
export OPENAI_API_KEY="sk-..."       # bash / zsh
# $env:OPENAI_API_KEY="sk-..."      # Windows PowerShell

# 4. Smoke-test the setup
python -c "from langchain_openai import ChatOpenAI; \
           r = ChatOpenAI(model='gpt-4o-mini').invoke('Say: ready'); \
           print(r.content)"

🎯 What You Will Learn

By completing this course you will be able to:

  • Explain how LLMs, RAG, and tool-use combine to create intelligent agents
  • Design and implement multi-agent pipelines using LangChain and LangGraph
  • Understand and apply the MCP and Agent-to-Agent (A2A) communication protocols
  • Build multimodal, specialized, and evaluation-aware agents
  • Apply system-design principles to architect scalable, production AI systems
  • Write Python code for GenAI applications end-to-end

🚀 Quick Start

Clone the repository, install dependencies, and run the multi-agent demo:

git clone <repo-url>
cd multi-agent-langgraph/code
pip install -r requirements.txt
export OPENAI_API_KEY="sk-..."
python main.py

For the interactive Streamlit demo:

cd demo/
bash run_demo.sh
