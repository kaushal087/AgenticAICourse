🤖 Build a Multi-Agent System with CrewAI & LangGraph

Interview Kickstart — Agentic AI Instructor Demo
Topic 4: Build a Multi-Agent System with CrewAI / AutoGen and LangGraph
Audience: Working professionals (5+ years SWE) transitioning to Generative AI
Session Duration: 30–45 minutes

🎯 What You'll Learn

By the end of this session, you will be able to:

  1. Explain why multi-agent systems outperform single-agent pipelines
  2. Understand LangGraph's stateful graph execution model
  3. Design agent roles: Planner, Researcher, Analyst, Writer
  4. Implement inter-agent communication and shared state
  5. Run a working multi-agent research assistant end to end

🚀 Quick Start

Prerequisites

  • Python 3.10+
  • OpenAI API key (set as OPENAI_API_KEY environment variable)

1. Install dependencies

cd code/
pip install -r requirements.txt

2. Set your API key

export OPENAI_API_KEY="sk-..."

3. Run the CLI demo

python main.py

4. Run the Streamlit demo

cd demo/
bash run_demo.sh

🧠 Key Concepts Covered

Concept Description
Multi-Agent Systems Independent agents collaborating on complex tasks
LangGraph State-machine graph for orchestrating agent workflows
Agent Roles Planner → Researcher → Analyst → Writer pipeline
Shared State GraphState TypedDict passed between nodes
Tool Use Web search, calculator, RAG retrieval
RAG Integration FAISS-based vector store for knowledge retrieval
Human-in-the-Loop Interrupt graph for supervisor approval

🛠 Tech Stack

  • LangGraph — graph-based agent orchestration
  • LangChain — LLM abstraction, tools, chains
  • OpenAI GPT-4o — reasoning backbone
  • FAISS — vector similarity search for RAG
  • Streamlit — interactive demo UI

📝 Instructor Notes

  • Assumption about students: Familiar with Python, REST APIs, and basic ML concepts. No prior LangChain/LangGraph experience assumed.
  • Teaching flow: Start with the analogy → draw the architecture → walk the code → run the demo → quiz.
  • Common confusion: Students often confuse "chains" (sequential) vs "graphs" (conditional branching). Emphasize this early.
  • Interview angle: Multi-agent system design is a common senior ML Engineer interview topic at FAANG.
