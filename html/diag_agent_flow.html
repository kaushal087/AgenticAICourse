Agent Execution Flow — Mermaid Source

sequenceDiagram
    actor User
    participant P as 🧠 Planner
    participant R as 🔍 Researcher
    participant A as 🧪 Analyst
    participant W as ✍️ Writer
    participant Tools as 🛠 Tools (Search + RAG)

    User->>P: "Analyze AI's impact on software jobs by 2026"

    Note over P: Build task decomposition prompt
    P->>P: GPT-4o call → JSON plan
    P-->>R: plan: ["Q1", "Q2", "Q3", "Q4", "Q5"]

    loop For each subtask in plan
        R->>Tools: search_web(subtask)
        Tools-->>R: web snippets
        R->>Tools: rag_retrieve(subtask)
        Tools-->>R: doc chunks
        R->>R: synthesize findings
    end
    R-->>A: research_results: [{subtask, findings}×5]

    Note over A: Evaluate coverage, identify gaps
    A->>A: GPT-4o call → coverage score

    alt coverage < 70% AND iteration < 3
        A-->>R: approved=false, additional_queries=["follow-up Q"]
        Note over R: Run again with follow-up queries
        R-->>A: updated research_results
    end

    A-->>W: analysis + approved=true

    Note over W: Build comprehensive report
    W->>W: GPT-4o call → markdown report
    W-->>User: # Final Report (800+ words, with citations)
